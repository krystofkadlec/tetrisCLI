# tetrisCLI
Simple tetris clone for linux terminal.

## Usage
  To compile, you can use gcc: `gcc src/*.c -o tetris`  
  
  Move the falling piece with arrow keys or `A` and `D`.  
  Rotate it clockwise with `W` or arrow up.  
  Soft-drop the piece with `S` or arrow down.  
  Use `Space` for hard drop.  
  To play again after gameover, press `Enter`
